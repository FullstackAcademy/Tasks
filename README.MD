Heya! This is my TaskFlow manager, I basically made this go in Sync with Google Calendar, you can export to an ics file (Apple has no APi ;/ btw this can sync with all of your google calendar, so you can see your events from google as well as export new events via the website.

Anywho, you can have tasks for the week, take notes to your hearts content and there are categories so that you can essentially break everything up and have your own perosnal touch to it by adding images (JPEG/JPG) 

Buncha accounts and such, this took so much time out , just being able to access the account was tedious but finally, here is the product!

STACKS USED:

Backend: Node, Express, Prisma, SQL, JSON web tokens, googleAPIS, ics
Frontend: React, Vite, Tailwind (Thanks Eliot for the Rec!), Flowbite, FullCalendar, Axios.
